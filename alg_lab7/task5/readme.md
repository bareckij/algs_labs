## Задание 5

Вычислить длину самой длинной общей подпоследовательности из трех последовательностей

- F0 = F1 = 1 (1)
- Fi = Fi−1 + Fi−2 для i ≥ 2

| Input | Output |
| ----- | ------ |
| 3     | 2      |
| 1 2 3 |
| 3     |
