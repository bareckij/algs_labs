## Запуск проекта

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/bareckij/algs_labs.git
   ```
2. Перейдите в папку с проектом:
   ```bash
   cd algs_labs
   ```
3. **Запуску всех задач**

   ```bash
   for script in alg_lab7/*/src/*.py; do PYTHONPATH=$(pwd) python3 "$script"; done

   ```
